# Cari Harta - Telegram Mini App

Game scavenger hunt 1000 barang rumah tangga

## Deploy ke Vercel (Backend)
1. Upload folder ini ke GitHub
2. Import ke Vercel
3. Deploy

## Deploy Frontend ke Netlify
1. Upload index.html ke Netlify
2. Set URL di BotFather: https://cari-harta.netlify.app

## API Endpoints
- GET /api/hunt - dapat misi aktif
- POST /api/submit - kirim foto
- GET /api/leaderboard - leaderboard
